Installation

Quick Installation

git clone https://github.com/johanlido/Synaptic.git
cd Synaptic
./scripts/interactive-setup.sh

The interactive setup script will guide you through:

  • Platform detection and prerequisites checking
  • MCP server installation and configuration
  • Claude Desktop integration setup
  • VS Code and GitHub Copilot configuration

Configuration

Environment Variables

Configure your API keys in the .env file:

ANTHROPIC_API_KEY=your_claude_api_key
PERPLEXITY_API_KEY=your_perplexity_key
FIGMA_ACCESS_TOKEN=your_figma_token

Claude Desktop Config

MCP servers are automatically configured in:

~/.claude_desktop_config.json

Restart Claude Desktop after configuration changes.

Best Practices

Security

  • Never commit API keys to version control
  • Use environment variables for sensitive data
  • Regularly rotate API keys
  • Follow WCAG 2.1 AA guidelines

Development

  • Use AI guardrails for consistent code quality
  • Test MCP server connections regularly
  • Monitor API usage and costs
  • Keep dependencies updated
