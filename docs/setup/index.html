Prerequisites

Required Software

  • Node.js 18+ and npm
  • Git version control
  • VS Code or preferred editor
  • Claude Desktop app

API Keys Needed

  • Anthropic API key (Claude)
  • Perplexity API key
  • Figma API token (optional)
  • GitHub Copilot subscription

Detailed Setup Steps

1

Install Prerequisites

Install the required software for your platform:

# macOS (using Homebrew)
brew install node git
brew install --cask claude visual-studio-code

# Windows (using Chocolatey)
choco install nodejs git vscode

# Linux (Ubuntu/Debian)
sudo apt update
sudo apt install nodejs npm git curl
2

Clone and Setup Repository

Get the synaptic repository and run the interactive setup:

git clone https://github.com/johanlido/Synaptic.git
cd Synaptic
chmod +x scripts/*.sh
./scripts/interactive-setup.sh
3

Configure API Keys

Set up your environment variables with API keys:

cp .env.example .env
# Edit .env file with your API keys:
# ANTHROPIC_API_KEY=your_claude_api_key
# PERPLEXITY_API_KEY=your_perplexity_key
# FIGMA_ACCESS_TOKEN=your_figma_token
4

Configure Claude Desktop

Set up MCP servers in Claude Desktop:

./scripts/configure-claude.sh
# This will update your Claude Desktop config
# Restart Claude Desktop after configuration
5

Verify Installation

Test that everything is working correctly:

# Test MCP servers
./scripts/test-setup.sh

# Open Claude Desktop and verify MCP servers are connected
# Open VS Code and check GitHub Copilot is active
